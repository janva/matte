Gränsvärden

standardgränsvärden $(x \to 0)$

\begin{align} \frac{\sin{x}}{x} &\to 1 \mbox { då } x \to 0\\ \frac{\tan{x}}{x} &\to 1 \mbox { då } x \to 0\\ \frac{\ln{1+x}}{x} &\to 1 \mbox { då } x \to 0\\ \frac{e^x-1}{x} &\to 1 \mbox { då } x \to 0\\ (1+x)^{\frac{1}{x}} &\to e \mbox { då } x \to 0\\ (x)^{\alpha}\ln{x} &\to 0 \mbox { då } x \to 0^{+}\\ \end{align}

Standardgränsvärden arcus $(x \to 0)$

\begin{equation} \frac{\arcsin{x}}{x} \to 1 \quad \mbox { och } \quad \frac{\arctan{x}}{x} \to 1 \end{equation}

Standardgränsvärden $(x \to \infty)$

\begin{align} \frac{\ln{x}}{x^\alpha} &\to 0 \mbox{ då } x\to \infty \mbox { och }\alpha > 0 \\ \frac{x^\alpha}{a^x} &\to 0 x\to \infty \mbox { och } a>1 \end{align}

standardgränsvärden $(n \to \infty)$ för heltal

\begin{equation} a^n \to \left\{ \begin{array}{l l} 0 & \quad \mbox{om} \mid a \mid < 1\\ 1 & \quad \mbox{om a=1 }\\ \infty & \quad \mbox{om a > 1 }\\ \end{array} \right. \\ \end{equation} \begin{align} \frac{ln{n}}{n^{\alpha}}& \to 0 \mbox{ om konstanten } \alpha > 0 \\[0.5mm] \frac{n^\alpha}{a^n} &\to 0 \mbox{ om konstanten} a > 1 \\[0.5mm] \frac{a^\alpha}{n!} &\to 0 \\[0.5mm] \frac{n!}{n^n} &\to 0 \\[0.5mm] \sqrt[n]{n^\alpha}& \to 0 \mbox{ för varje } \alpha \end{align}
