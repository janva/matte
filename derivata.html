Derivata

Derivatans definition

En funktion f är Deriverbar i en punkt a, om f är definierad i någon omgivning av a och gränsvärdet. \begin{equation} \lim_{x \to a}\frac{f(x)-f(a)}{x-a} \end{equation} existerar ändligt. ersätter vi x med \((a + h)\) får vi. \begin{equation} \lim_{h \to 0}\frac{f(a+h)-f(a)}{h} \end{equation}

Räkneregler

\begin{align} \frac{d}{dx}&\left(cf(x)\right) = cf\prime(x)\\ \frac{d}{dx}&\left(f(x)+g(x)\right) = f\prime(x)+g\prime(x)\\ \frac{d}{dx}&\left(f(x)g(x)\right) = f\prime(x)g(x)+f(x)g\prime(x)\\ \frac{d}{dx}&\left(\frac{f(x)}{g(x)}\right) = \frac{f\prime(x)g(x)-f(x)g\prime(x)}{(g(x))^2} \end{align}

Kedjeregeln

För sammansättningen av två funktioner f och g gäller den s k kedjergeln. \begin{equation} \frac{d}{dx}(f(g(x)))=f\prime(y)g\prime(x) \mbox{med y = g(x)} \end{equation} om g är deriverbar i x och f är deriverbar i y = g(x).

Standard derivator

\begin{align} \frac{d}{dx}&(\ln{x})=\frac{1}{x} \mbox{ för }x > 0 \\ \frac{d}{dx}&(e^x)=e^x \mbox{ för alla reella x}\\ \frac{d}{dx}&(\ln{\mid x\mid})=\frac{1}{x} \mbox{ för }x \neq 0\\ \frac{d}{dx}&(x^{\alpha})=\alpha x^{\alpha-1}\mbox{ för }x > 0 \mbox{ och alla konstanter } \alpha\\ \frac{d}{dx}&(a^x)=a^x\ln{a} \mbox { för alla rella x }\neq 0\\ \frac{d}{dx}&(log_a\mid x\mid)= \frac{1}{x\ln{a}} \mbox{ för alla reella }x > 0 \mbox{ med }a \neq 1 \end{align}

Trigonometriska standardderivator

\begin{align} \frac{d}{dx}&(\sin{x})=\cos{x}\\ \frac{d}{dx}&(\cos{x})=-\sin{x} \\ \frac{d}{dx}&(\tan{x})=\frac{1}{\cos^2{x}}=1+\tan^2{x}\\ \frac{d}{dx}&(\cot{x})=\frac{1}{\sin^2{x}}=-1+\cot^2{x} \\ \end{align}

Standardderivator arcus

\begin{align} \frac{d}{dx}&(\arcsin{x})=\frac{1}{\sqrt{1-x^2}} \mbox { för } -1 < x < 1\\ \frac{d}{dx}&(\arccos{x})=-\frac{1}{\sqrt{1-x^2}} \mbox { för } -1 < x < 1\\ \frac{d}{dx}&(\arctan{x})=-\frac{1}{1+x^2} \mbox { för alla reella } x\\ \end{align}
